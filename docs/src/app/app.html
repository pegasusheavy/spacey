Why Spacey?

Built from the ground up with modern web platform needs in mind

Memory Safe

Leverages Rust's ownership model for a JavaScript runtime free of memory bugs and data races.

Pure Rust

No C++ dependencies. Enables tighter integration with Rust-based projects like Servo.

Modular Design

Clean architecture with separate crates for engine, runtime, and Node.js compatibility.

TypeScript Native

Direct TypeScript parsing without transpilation. Types are stripped at parse time for fast execution.

Generational GC

High-performance garbage collector with nursery/old generation, incremental marking, and parallel sweeping.

Node.js Compatible

Full Node.js API layer with CommonJS, ES Modules, and async I/O through Tokio.

Architecture

A modern, modular design inspired by decades of JavaScript engine research

Getting Started

Get up and running with Spacey in minutes

1 Build from Source

terminal 
# Clone the repository
git clone https://github.com/pegasusheavy/spacey.git
cd spacey

# Build all crates
cargo build --release

# Run the REPL
cargo run

2 Use as a Library

main.rs
